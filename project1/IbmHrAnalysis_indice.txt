Análise de RH - Rotatividade e Desempenho de ColaboradoresPedro Guerreiro/Gonçalo Branco/Sina Sarikhani Equipa Numero 5Analisar o dataset 'IBM HR Analytics Employee Attrition & Performance' para identificar fatores que contribuem para a rotatividade e oportunidades para melhorar a retenção e o desempenho.Objetivos Principais de Projeto:• Carregar e Analisar o dataset de dados de projeto• Realizar Análise estatística descritiva de Dados com R em VS-Code• Normalização base de dados (Até 3NF)• Inserir dados em SQL e fazer consultas• Identificar um caso de uso para base de dados NoSQL (MongoDB)• Esquematizar preparação de dados (Power Query) e modelagem OLAP (Power BI).• Definir questões de negócio e indicadores analíticos.• Propor conclusões e recomendações baseadas em dados.Índice :O projeto incluindo 6 pastas e 1 ficheiro PPTX. As pastas incluindo:• 1- Dictionay: um ficheiro xlsx - Dicionário de Dados do Dataset• 2-R:  3 ficheiros - Resumo da Análise Exploratória de Dados com VS-Code :o um ficheiro cmd com o scrito de trabalhoo 2 ficheiros html e pdf incluindo o resultado de análise• 3-Normalizaçao:  Processo de Normalização da Base de Dados (3NF) em xlsx• 4-SOL: Um ficheiro sql de scrito de Implementação da Base de Dados• 5-Mongodb:  Um ficheiro Json de scrito e um ficheiro de docx de consultas - Caso de Uso NoSQL (MongoDB)• 6-PowerBi: Um ficheiro pbix de projeto feito no PowerBi incluindo:o Comparação visual dos dados antes e depois das transformaçõeso Imagem do modelo OLAP em estrelao Imagens das principais visualizaçõesE um ficheiro xlsx:                Lista das métricas criadas, com a respetiva explicação.
